## 数据流宽度

```
DDS (64bit phase) → 12bit LUT addr → 16bit sample
    → amp_offset_scale (16bit × 16bit = 32bit → 16bit饱和)
    → JESD204 Transport (16bit × M=2 = 32bit/frame)
    → 8B/10B编码 → 4Lane × 10Gbps serial
```

## 与教学DAC版本的对比

| 项目 | awg_dds_led_top (教学) | awg_fmc_adda_top (目标) |
|---|---|---|
| DAC | ATK-HS-ADDA (8bit并行) | AD9144 (JESD204B 16bit) |
| 采样率 | ~100MHz (并口) | 2.8Gsps (JESD204B) |
| DDS | Xilinx IP (48bit) | 手写NCO (64bit) |
| 波形 | 正弦 | 正弦/方波/三角/锯齿/任意 |
| 调节 | 无 | 幅度+偏置+饱和 |
| 输出 | LED+DAC | JESD204B高速串行 |
