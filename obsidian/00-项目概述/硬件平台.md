| 资源 | 规格 |
|---|---|
| FPGA | XC7K325TFFG900-2I |
| 封装 | FFG900 |
| 速度等级 | -2 |
| 核心板时钟 | 100MHz 有源差分晶振 (AE10/AF10) |
| 核心板 DDR3 | 4×4Gbit NT5CC256M16, 共 2GB |
| 核心板 Flash | N25Q128, 128Mbit / 16MB |

## FMC ADDA 子卡 (FMCADDA-9250-9144)

| 芯片 | 型号 | 规格 | 接口 | 用途 |
|---|---|---|---|---|
| **DAC** | AD9144 | 4ch, 16bit, 2.8Gsps | JESD204B (8Lane max) | 高速波形输出 |
| **ADC** | AD9250 | 2ch, 14bit, 250Msps | JESD204B (2Lane) | 信号采集回环 |
| **时钟** | LMK04828 | 双环路 PLL | SPI | 系统时钟树 |
| **参考** | 50M TCXO | 0.3ppm | — | LMK 输入参考 |

### AD9144 目标配置

```
Mode: 4 Lane (Subclass 1)
Lane Rate: 10 Gbps
F = 1 (Octets/Frame/Lane)
K = 32 (Frames/Multiframe)
M = 2 (Converters, DAC0+DAC1)
S = 1 (Samples/Converter/Frame)
NP = 16 (Bits/Sample)
Scrambling: Enabled
```

### 时钟网络

```
50M TCXO → LMK04828
    ├── OUT0  = 125M  → FPGA GTX REFCLK
    ├── OUT8  = 500M  → AD9144 DAC CLK
    └── OUT1  = 3.90625M → SYSREF (Subclass 1)
```

### FMC HPC 引脚映射 (已提取)

详见 [[FMC引脚约束]] 或 `constraints/fmc_adda.xdc`。

关键差分对：
- **DAC TX (C2M)**: DP0~DP3 (H2/H1, F2/F1, J4/J3, K2/K1)
- **ADC RX (M2C)**: DP0~DP1 (G4/G3, F6/F5)
- **GTX REFCLK**: GBTCLK0 (G8/G7)
- **SYSREF**: LA00_CC (D17/D18)
- **SPI**: LAxx 系列 (LMK + AD9144)
