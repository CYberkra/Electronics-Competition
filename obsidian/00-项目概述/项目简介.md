# 项目简介

## 竞赛背景

**第二十一届研电赛优利德赛题二** —— 5GSa/s 任意波形发生器（AWG）

> FPGA 作为数字基带/控制平台，配合外部高速 DAC、低抖动时钟、重建滤波器实现 5GSa/s、14bit、1GHz 模拟带宽的波形输出。

## 核心指标

| 指标 | 目标值 | FPGA 侧职责 |
|---|---|---|
| 采样率 | 5 GSa/s | JESD204B 8Lane 数据分发 |
| 分辨率 | 14 bit | 16bit 数据 → DAC（AD9144） |
| 模拟带宽 | 1 GHz | 数字域预补偿（可选） |
| 频率分辨率 | 1 mHz | 64bit DDS NCO |
| 输出幅度 | 可调 | `amp_offset_scale.v` 定点乘法 |
| 波形类型 | 正弦/方波/三角/锯齿/任意 | 多波形生成 + BRAM/DDR3 回放 |
| 扫频功能 | 线性/对数扫频 | `sweep_engine` 模块 |

## 当前聚焦

**阶段一：单路 DAC0 输出 10MHz 正弦**
- 建立 JESD204B 链路（AD9144 4L, 10Gbps）
- LMK04828 配时钟 → AD9144 SPI 初始化 → FPGA 发波形
- 适配学校 20~50MHz 带宽示波器验证
