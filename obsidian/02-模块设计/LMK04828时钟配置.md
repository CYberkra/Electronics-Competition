# LMK04828 时钟配置

## 芯片概述

LMK04828 是双环路时钟抖动清除器，用于从 50M TCXO 产生 JESD204B 系统所需的所有时钟。

## 输出时钟分配

| 输出 | 频率 | 用途 | 目标 |
|---|---|---|---|
| OUT0 | 125 MHz | FPGA GTX REFCLK | FPGA GBTCLK0 (G8/G7) |
| OUT1 | 3.90625 MHz | SYSREF | FPGA + AD9144 (Subclass1) |
| OUT8 | 500 MHz | DAC CLK | AD9144 CLK+/- |
| 其他 | — | 未使用 | — |

## 配置来源

寄存器表来自子卡资料：
```
HexReg_9250_9144_04828_125M_500M_gen.txt
```

包含 80+ 条配置（R0~R395, R8189~R8191）。

## RTL 实现

**文件**: `rtl/clk/lmk04828_spi_ctrl.v`

### 状态机

```
IDLE ──→ WRITE_REG ──→ WAIT_DONE ──→ CHECK_LOCK ──→ DONE
            ↑___________________________________________|
```

### 接口

```verilog
module lmk04828_spi_ctrl (
    input        clk,       // 100M system clock
    input        rst_n,
    output reg   spi_csb,   // SPI Chip Select (active low)
    output reg   spi_sclk,  // SPI Clock
    inout        spi_sdio,  // SPI Data (bidirectional)
    output reg   locked,    // PLL locked indicator
    output reg   done       // Configuration complete
);
```

### SPI 时序

- 模式: SPI Mode 0 (CPOL=0, CPHA=0)
- 时钟: ~1MHz (100M 系统时钟分频)
- 数据: 24bit (R/W + W1+W0 + Addr[12:0] + Data[7:0])

### 关键寄存器

| 地址 | 值 | 功能 |
|---|---|---|
| R0 | 0x80 | Reset device |
| R1-R395 | varies | PLL1/PLL2 配置、分频器、输出使能 |
| R0x00 | SPI config | 3-wire/4-wire mode |
| R0x01 | 0x02 | Device ID |

## 上电时序

```
FPGA 上电
    ↓
FPGA 释放 sys_rst_n
    ↓
lmk04828_spi_ctrl 自动开始 (start = 1'b1)
    ↓
按寄存器表顺序写入全部 80+ 个寄存器
    ↓
等待 PLL1_LOCK + PLL2_LOCK
    ↓
done = 1, locked = 1
    ↓
AD9144 SPI 开始配置
```

## 验证方法

1. **LED 指示**: locked 信号可接 LED，亮表示时钟就绪
2. **示波器**: 测量 LMK OUT0/OUT1/OUT8 输出频率
3. **FPGA ILA**: 在线抓取 spi_csb/sclk/sdio 波形，确认寄存器写入时序

## 注意事项

1. LMK04828 的 SPI 是 **3-wire 模式**（SDIO 双向），需要 inout 处理
2. 配置完成后 **必须等待 PLL 锁定**，否则后续 JESD204 建链必失败
3. SYSREF 输出建议配置为 **gapped periodic**（脉冲模式），而非连续时钟
4. 50M TCXO 输入到 LMK 的参考路径不能过长，保证信号完整性
