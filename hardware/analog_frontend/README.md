# AWG模拟前端设计说明

## 项目概述

本设计为K325T + AD9144 AWG项目的模拟前端模块，用于将AD9144 DAC输出信号进行增益控制、量程切换和50Ω匹配输出。

## 技术指标

| 参数 | 目标值 | 说明 |
|------|--------|------|
| 输入 | AD9144子卡SMA输出 | 差分或单端，约2.7Vpp |
| 输出幅度范围 | 10mVpp ~ 3Vpp | 满足赛题要求 |
| 输出阻抗 | 50Ω | SMA接口 |
| 带宽 | ≥1GHz | -3dB |
| 控制接口 | SPI + GPIO | FPGA控制 |

## 电路拓扑

```
SMA输入(J1) → 巴伦(T1) → 固定增益放大(U1) → 数控衰减(U2) → 量程切换(K1) → 输出驱动(U3) → SMA输出(J2)
```

## 关键器件

### T1: TC1-1-13M+ (巴伦)
- 功能：差分转单端（如果子卡输出是差分）
- 带宽：4.5MHz ~ 3GHz
- 阻抗比：1:1

### U1: THS3491 (固定增益放大)
- 功能：将AD9144输出提升到足够电平
- 带宽：900MHz
- 增益：+6dB（约2倍）
- 供电：±5.5V

### U2: PE4302 (数控衰减器)
- 功能：精确控制输出幅度
- 衰减范围：0 ~ -31.5dB
- 步进：0.5dB
- 控制：并行数字接口（D0-D5 + LE + CLK）
- 带宽：1MHz ~ 4GHz

### K1: G6K-2F (量程切换继电器)
- 功能：切换高/低/超低量程
- 触点：DPDT
- 带宽：2GHz
- 线圈：5V

### U3: THS3491 (输出驱动)
- 功能：50Ω输出驱动
- 带宽：900MHz
- 增益：单位增益（或根据需要调整）

## 量程设计

| 量程 | 继电器状态 | 衰减网络 | 输出范围 |
|------|-----------|---------|---------|
| 高量程 | K1A闭合 | 0dB | 0.1V ~ 3Vpp |
| 低量程 | K1B闭合 | -20dB | 10mV ~ 300mVpp |
| 超低量程 | K1C闭合 | -40dB | 1mV ~ 30mVpp |

## 电源需求

- +5V @ 200mA（运放供电）
- -5V @ 100mA（运放负电源）
- +3.3V @ 50mA（PE4302逻辑、继电器驱动）

## FPGA控制接口

```
FPGA GPIO:
  - RANGE_SEL[1:0]: 量程选择（00=高, 01=低, 10=超低）
  - ATT_LE: PE4302锁存使能
  - ATT_CLK: PE4302时钟
  - ATT_D[5:0]: PE4302衰减数据
  - OUTPUT_EN: 输出使能
```

## 设计要点

1. **阻抗控制**：所有RF走线必须保持50Ω特性阻抗
2. **地平面**：完整的射频地平面，避免割裂
3. **去耦**：每个电源引脚0.1μF + 10μF陶瓷电容
4. **布局**：信号路径短直，继电器靠近输出端
5. **屏蔽**：高频部分加金属屏蔽罩
