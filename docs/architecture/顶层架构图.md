## 时钟树

```
50M TCXO (板载)
    │
    ▼
LMK04828 PLL
    │
    ├── OUT4 (125MHz) ──→ GBTCLK0 ──→ GTX QPLL RefClk
    ├── OUT6 (125MHz) ──→ FMC glblclk ──→ clk_for_glbclk MMCM
    │                                       │
    │                                       ├── w_rx_core_clk (125MHz)
    │                                       └── w_tx_core_clk (250MHz)
    └── OUT9 (SYSREF) ──→ LA20_P/N ──→ jesd204 SYSREF

100M 差分 (板载)
    │
    ▼
IBUFDS → BUFG → sys_clk_bufg (100MHz)
    │
    ▼
STARTUPE2 → CFGMCLK (65MHz)
    │
    ▼
clk_sys_mmcm MMCM
    │
    ├── clk_25m     (SPI 控制时钟)
    └── clk_axi_100m (AXI-Lite 配置时钟)
```

## SPI 控制架构

```
                      clk_25m
                         │
          ┌──────────────┼──────────────┐
          │              │              │
          ▼              ▼              ▼
  lmk_spi_wr_config  ad9250_spi_config  ad9144_spi_wr_config
  (LMK04828)         (AD9250 ADC)       (AD9144 DAC)
          │              │              │
          ▼              ▼              ▼
      spi_wr_rd_single (通用 SPI 控制器)
          │
          ├── o_sclk / o_sda / o_cs_n   (三线 SPI)
          └── i_sda                      (半双工数据)
```

所有 3 个 SPI 实例共享同一个 `spi_wr_rd_single` 模块，25MHz 时钟域。

## 初始化顺序

上电后状态机的执行序列：

```
Step 1: 等待 CFGMCLK (65MHz) 稳定
    │
Step 2: 配置 LMK04828 (SPI 写 PLL 分频/输出)
    │
Step 3: 配置 AD9250 ADC (SPI)
    │
Step 4: JESD204B TX 复位 → 等待 GTX PLL Lock
    │
Step 5: AXI-Lite 配置 JESD204 TX/RX IP
    │
Step 6: 配置 AD9144 DAC (SPI)
    │
Step 7: JESD204B RX 复位 → 等待链路建链
    │
Step 8: DDS 开始输出 (Tone 产生)
```

## 控制方式

| 方式 | 接口 | 优先级 | 说明 |
|------|------|--------|------|
| 板载按键 | KEY0/KEY1 | 低 | 频率/幅度/偏置/波形切换 |
| UART | 115200 8N1 | 中 | 通过 `ad9144_uart_reg_bridge` 读写寄存器 |
| 寄存器组合 | `awg_reg_use_control` | 高 | UART 使能时覆盖按键控制 |

## 关键信号路径

```
DDS 数据路径 (250MHz tx_core_clk):
  phase_acc[47:0] → phase+offset → addr[11:0] → sine_rom[4096]
    → wave_shape_mux → DSP48 multiply(×amplitude) → scale+sat → sample[15:0]
    → packer → tx_tdata[127:0] → JESD204 TX → GTX → DAC

SPI 控制路径 (25MHz clk_25m):
  state_machine → spi_wr_rd_single → o_sclk + o_sda + o_cs_n → 目标芯片
```
