## 1. 采样率与带宽测试

**测量说明**：
- 采样率测量：使用频谱仪近场探头耦合 DAC 时钟信号，或通过 SMA 测试点直接测量。Center = 预期时钟频率 (1.0~1.5 GHz)，Span = 1 MHz，RBW = 1 kHz。Marker Peak Search 读取频率，根据 DDR/SDR 模式换算采样率：Fs = f_DACCLK × 2 (DDR) 或 Fs = f_DACCLK (SDR)。
- 带宽测量：AWG 输出固定幅度正弦波 (如 0 dBm)，从参考频率 (10 MHz) 开始记录功率 P_ref。按步进增加频率（保持幅度设置不变），频谱仪 Center 跟踪 f_out，记录每个频点的峰值功率 P(f)。-3 dB 带宽定义为 P(f) - P_ref ≤ -3.0 dB 的最高频率。

## 2. 垂直分辨率测试 (SINAD/ENOB 方法)

**测量说明**：
- 依据命题方指定的 SINAD 方法。核心公式：ENOB = (SINAD - 1.76) / 6.02。
- 测试流程：
  1. AWG 输出指定频率正弦波，最大不失真幅度 (~3 Vpp, ~13 dBm @ 50Ω)。
  2. 频谱仪 Center = f_out，Span 覆盖基波及 5 次谐波。记录基波功率 P_signal (dBm)。
  3. NSD 测量：频谱仪 Span 展宽至 50 MHz，Marker 放置于远离载波 ≥ 1 MHz 的噪声平坦区，Marker Noise 模式读取 NSD (dBm/Hz)。
  4. 谐波测量：依次定位 2nd, 3rd, 4th, 5th 谐波，记录各谐波功率 P_harm(n) (dBm)。
  5. 计算步骤：
     - P_noise = NSD + 10 × log10(Fs / 2)
     - SNR = P_signal - P_noise
     - THD = 20 × log10( √(Σ 10^(P_harm(n)/10)) / 10^(P_signal/20) )
     - SINAD = -10 × log10( 10^(-SNR/10) + 10^(-THD/10) )
     - ENOB = (SINAD - 1.76) / 6.02
- 至少测试 8 个频率点 (1 MHz ~ 400 MHz)，记录最差 ENOB。

## 6. 输出幅度测试

**测量说明**：
- 命题方要求：输出幅度范围 10 mVpp ~ 3 Vpp (50 Ω 负载)，幅度精度 ≤ 1 mVpp。
- 推荐方法：功率计法用于精确幅度校准，频谱仪法用于辅助验证。
- 功率计法步骤：AWG 输出 10 MHz 正弦波，依次设置目标幅度。功率计预热、归零、校准后测量，每个目标取 3 次读数平均。绝对误差 = |实测幅度 - 目标幅度|。
- 功率与幅度换算 (50 Ω)：P(dBm) = 10 × log10(Vpp² / 0.4)，其中 Vpp 单位伏特。
- 50 Ω 端接必须确认：频谱仪/功率计输入端设为 50 Ω 匹配模式。

## 9. 输出阻抗验证 (可选)

**测量说明**：
- 验证 AWG 输出阻抗是否为 50 Ω。
- 方法一 (VNA 法)：VNA 单端口 SOL 校准后，测 S11。回波损耗 RL = -20 log10(|S11|)。S11 < -15 dB @ DC~1 GHz 对应 VSWR < 1.43，即 |Z| 在 35~71 Ω 之间，可认为匹配良好。
