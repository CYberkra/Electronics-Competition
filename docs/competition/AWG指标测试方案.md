## 2. 采样率与输出带宽测试

### 2.1 测试目的与指标要求

| 指标 | 赛题要求 | 本方案设计值 | 备注 |
|------|----------|-------------|------|
| DAC 采样率 | ≥ 5 GSa/s | 2.8 GSa/s (AD9144 上限) | 硬件限制, 架构可迁移至更高速率 DAC |
| 模拟输出带宽 | ≥ 1 GHz | 待测试 (~800 MHz @ 2.8G) | Nyquist 带宽 = Fs/2 = 1.4 GHz (理论) |

### 2.2 测试原理

#### 2.2.1 采样率测试 (DAC 时钟频率测量)

根据命题方指定方法：使用频谱仪直接测量 DAC 时钟输入端，检测主时钟频率。

AD9144 的 DAC 采样时钟 (DACCLK) 由 LMK04828 提供。在子卡设计中，DACCLK 通常通过 SMA 测试点或耦合路径可被频谱仪探测。

**替代测量方案** (当 DAC 时钟输入端不可直接接入时)：

1. **通过输出信号反推**：设置 AWG 输出已知频率正弦波，频谱仪观察 Nyquist 镜像位置
   - DAC 采样率 `Fs = f_Nyquist_mirror - f_out` 或 `f_Nyquist_mirror + f_out`
   - 示例：输出 100 MHz，若在 Fs-100 MHz 处看到镜像，则 Fs = f_mirror + 100 MHz
2. **通过 JESD204B lane rate 反推**：已知 lane rate = 10 Gbps
   - `f_DAC = (lane_rate × 4 lanes × 8/10) / (M × S × 内插因子)`
   - `f_DAC = (10G × 4 × 0.8) / (4 × 1 × 4) = 2.0 GHz`? (需根据实际 Mode 配置计算)

#### 2.2.2 输出带宽测试

定义 **-3 dB 带宽**：输出功率低于参考频率 (通常 1 MHz 或 10 MHz) 输出功率 3 dB 的最高频率。

### 2.3 测试步骤

#### 2.3.1 采样率确认

| 步骤 | 操作 | 参数设置 |
|:----:|------|---------|
| 1 | 频谱仪通过近场探头 (或 SMA 连接) 耦合 DAC 时钟信号 | 衰减器 ≥ 20 dB (防止过载) |
| 2 | 频谱仪中心频率设为预期 DAC 时钟频率 | Center = 1.0 GHz ~ 1.5 GHz |
| 3 | 频谱仪扫宽 1 MHz，RBW 1 kHz | Span = 1 MHz, RBW = 1 kHz |
| 4 | Marker → Peak Search 读取频率 | 记录峰值频率 = **DAC 采样时钟 f_DACCLK** |
| 5 | 计算 DAC 采样率 | **Fs = f_DACCLK × 2 (DDR 模式)** 或 **Fs = f_DACCLK (SDR 模式)** |

**频谱仪设置参考** (按命题方方法)：

```
Center Frequency: 5 GHz  (覆盖到 DAC 时钟可能的高频镜像)
Span:             1 MHz   (精确分辨时钟频率)
RBW:              1 kHz   (足够的频率分辨率)
VBW:              100 Hz  (降低噪声)
Sweep Time:       Auto
Reference Level:  0 dBm
Attenuator:       20 dB (先设高衰减，保护前级)
```

> 若 AD9144 使用内部 PLL (DAC PLL 倍频模式)，则测量 DACCLK 输入 (参考时钟) 并乘以内部倍频因子。

#### 2.3.2 输出带宽 (扫频法)

| 步骤 | 操作 | 参数设置 |
|:----:|------|---------|
| 1 | AWG 设置输出固定幅度正弦波 | 频率 = 1 MHz, 幅度 = 0 dBm (或 1 Vpp) |
| 2 | 频谱仪设置参考 | Center = 1 MHz, Span = 5 MHz, RBW = 10 kHz |
| 3 | 记录参考功率 `P_ref` (dBm) | Marker → Peak |
| 4 | 按步进增加 AWG 输出频率，保持幅度设置不变 | 步进建议：1M, 5M, 10M, 20M, 50M, 100M, 200M, 300M, 400M, 500M, 600M, 700M, 800M, 900M, 1G, 1.2G, 1.4G |
| 5 | 每个频点：频谱仪 Center = f_out, 记录峰值功率 `P(f)` | Span 随频率适当调整 |
| 6 | 计算各频点相对衰减 | `Attenuation(f) = P(f) - P_ref` |
| 7 | 确定 -3 dB 带宽点 | 找到 Attenuation(f) ≤ -3.0 dB 的最高频率 |

## 3. 垂直分辨率测试 (SINAD/ENOB 方法)

### 3.1 测试目的与指标要求

| 指标 | 赛题要求 | 本方案设计值 | 测试方法 |
|------|---------|-------------|---------|
| 垂直分辨率 | ≥ 14 bit | 16 bit (DAC 物理位宽) | SINAD → ENOB 换算 |
| 有效位数 (ENOB) | ≥ 11.5 bit (等效) | 待测试 | ENOB = (SINAD - 1.76) / 6.02 |

> **ENOB ≥ 11.5 bit 对应关系**：对于 16 bit 理想 DAC，ENOB 受限于实际 SINAD。SINAD ≥ 71 dB 时 ENOB ≥ 11.5 bit。

### 3.2 测试方法原理

根据命题方指定的 SINAD 方法：

**核心公式**：

```
(1) 信纳比与有效位数关系:
    SINAD (dB) = 6.02 × ENOB + 1.76
    → ENOB = (SINAD - 1.76) / 6.02

(2) 信纳比、信噪比、总谐波失真关系:
    SINAD = -10 × log10(10^(-SNR/10) + 10^(-THD/10))

(3) 信噪比计算:
    SNR (dB) = 信号功率(dBm) - 噪声功率(dBm)

(4) 总谐波失真计算:
    THD (dB) = 20 × log10( √(V2² + V3² + ... + Vn²) / V1 )
    其中 V1 为基波电压 RMS 值, V2~Vn 为 2~n 次谐波电压 RMS 值

(5) 噪声功率计算:
    噪声功率(dBm) = NSD + 10 × log10(Fs / 2)
    其中 NSD 为噪声谱密度 (dBm/Hz), Fs 为 DAC 采样率 (Hz)
```

**NSD 测量方法** (命题方指定)：

```
频谱仪中心频率 = DAC 输出信号频率
Marker 放置于远离载波 1 MHz 以上的噪声平坦区域
Marker → Noise 模式 (dBm/Hz)
读取 NSD 值
```

### 3.3 测试步骤

#### 3.3.1 基础设置

| 2 | 确认输出负载为 50 Ω | 频谱仪输入阻抗 = 50 Ω |
| 3 | 频谱仪设置 | Center = 10 MHz, Span = 500 kHz, RBW = 1 kHz, VBW = 100 Hz |
| 4 | 验证无过载或压缩 | 增加 1 dB 输入衰减确认峰值同步下降 1 dB |

### 3.4 计算公式与示例

#### 3.4.1 计算步骤

```
步骤 1 - 计算 SNR:
    噪声功率 P_noise = NSD + 10 × log10(Fs / 2)
    SNR = P_signal - P_noise
    (均以 dBm 为单位直接相减)

步骤 2 - 计算 THD:
    THD = 20 × log10( √(10^(P_h2/10) + 10^(P_h3/10) + ... + 10^(P_hn/10)) / 10^(P_signal/20) )
    其中 P_h2, P_h3, ... 为谐波功率 (dBm)
    简化: 先转为电压 RMS, 再计算

步骤 3 - 计算 SINAD:
    SINAD = -10 × log10( 10^(-SNR/10) + 10^(-THD/10) )

步骤 4 - 计算 ENOB:
    ENOB = (SINAD - 1.76) / 6.02
```

#### 3.4.2 计算示例 (假设值)

假设在 10 MHz 频率点测得：

| 参数 | 测量值 |
|------|--------|
| 基波功率 `P_signal` | +10.0 dBm |
| NSD @ 载波 +2 MHz | -150 dBm/Hz |
| DAC 采样率 `Fs` | 2.8 × 10^9 Hz |
| 2nd 谐波 (20 MHz) | -65 dBc |
| 3rd 谐波 (30 MHz) | -70 dBc |
| 4th 谐波 (40 MHz) | -75 dBc |
| 5th 谐波 (50 MHz) | -80 dBc |

```
计算:
  P_noise = -150 + 10 × log10(2.8e9 / 2)
          = -150 + 10 × log10(1.4e9)
          = -150 + 10 × 9.146
          = -150 + 91.46
          = -58.54 dBm

  SNR = 10.0 - (-58.54) = 68.54 dB

  谐波电压 RMS (相对基波):
    V2/V1 = 10^(-65/20) = 5.62e-4
    V3/V1 = 10^(-70/20) = 3.16e-4
    V4/V1 = 10^(-75/20) = 1.78e-4
    V5/V1 = 10^(-80/20) = 1.00e-4

  THD = 20 × log10( √(5.62e-4² + 3.16e-4² + 1.78e-4² + 1.00e-4²) )
      = 20 × log10( √(3.16e-7 + 1.00e-7 + 3.17e-8 + 1.00e-8) )
      = 20 × log10( √(4.48e-7) )
      = 20 × log10( 6.69e-4 )
      = 20 × (-3.17)
      = -63.4 dB
  |THD| = 63.4 dB  (> 40 dB 要求)

  SINAD = -10 × log10( 10^(-68.54/10) + 10^(-63.4/10) )
        = -10 × log10( 1.40e-7 + 4.57e-7 )
        = -10 × log10( 5.97e-7 )
        = -10 × (-6.224)
        = 62.24 dB

  ENOB = (62.24 - 1.76) / 6.02
       = 60.48 / 6.02
       = 10.05 bit
```

> 此示例 ENOB 未达 11.5 要求，说明需要降低噪声 (改善 NSD) 或降低谐波失真。NSD 从 -150 改善到 -155 可使 SINAD 提升 ~5 dB。

**频谱仪设置参考**：

```
Mode:        Spectrum Analyzer
Trace:       Max Hold (持续积累)
Center:      50.5 MHz
Span:        110 MHz (覆盖 START 到 STOP + 余量)
RBW:         300 kHz (大于步进 1 MHz 的 1/3)
VBW:         30 kHz
Sweep Time:  20 s (≥ 2 × 往返扫频总时间)
Attenuator:  Auto
```

**扫频总时间计算**：
```
频率点数 = (STOP - START) / STEP + 1 = (100M - 1M) / 1M + 1 = 100 点
单次扫频时间 = 100 点 × 10 ms = 1 s
往返时间 = 2 s
```

## 附录 A：频率控制字计算速查

```
phase_inc = f_out × 2^48 / f_clk

其中:
  f_out  = 期望输出频率 (Hz)
  f_clk  = tx_core_clk = 250 MHz (每个周期 4 采样, 等效 1 GHz 数据率)
  2^48   = 281,474,976,710,656

工程计算快捷值:
  phase_inc_per_Hz = 2^48 / 250e6 ≈ 1,125,899.906

常用频率 phase_inc 值:
  1 mHz:   0x000000000000 ~ 0
  1 Hz:    phase_inc = 1,126  → 0x000000000466
  1 kHz:   phase_inc = 1,125,900 → 0x000000112D6C
  1 MHz:   phase_inc = 1,125,899,907 → 0x0000431BDE83
  10 MHz:  phase_inc = 11,258,999,068 → 0x00029F16891C
  50 MHz:  0x000CCCCCCCCCD
  100 MHz: 0x001999999999A
  500 MHz: 0x0080000000000
  1 GHz:   0x0100000000000

注意: 当 f_out ≥ Fs/2 (Nyquist 极限) 时, 会进入第二 Nyquist 区,
     输出频率表现为 Fs - f_out 的镜像, 需配合带通滤波器使用
```
