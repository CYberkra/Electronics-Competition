## 技术备注

### 拍摄准备清单

**硬件准备：**
- [ ] FPGA开发板已上电
- [ ] FMC子卡正确安装
- [ ] 示波器探头连接OUT1
- [ ] UART转USB模块连接PC
- [ ] 50欧姆负载连接输出端口

**软件准备：**
- [ ] PC软件已启动
- [ ] 串口已识别(COM7)
- [ ] 设备已连接并初始化
- [ ] 默认参数：50MHz正弦波，幅度0x6000

**拍摄参数建议：**
- 示波器时基：根据频率调整
  - 10MHz: 50ns/div
  - 50MHz: 10ns/div
  - 100MHz: 5ns/div
- 示波器幅度：500mV/div
- 触发模式：Auto

### 应急方案

**情况1：设备无法连接**
- 检查UART线缆
- 确认COM口号
- 重新上电FPGA
- 重新编程bit文件

**情况2：波形异常**
- 检查JESD链路状态
- 确认DAC初始化完成
- 等待12-15秒初始化时间
- 重新设置参数

**情况3：软件崩溃**
- 重启Python程序
- 检查pyserial安装
- 确认串口未被占用
